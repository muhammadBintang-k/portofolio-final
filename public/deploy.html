Tutorial Deploy Web ke GitHub

Langkah 1: Buat Repository di GitHub

1. Masuk ke akun GitHub Anda, lalu klik tombol "New" untuk membuat repository baru.

2. Beri nama repository Anda dan pastikan untuk memilih opsi Public.

3. Klik Create Repository untuk melanjutkan.

Langkah 2: Upload Project ke Repository

1. Buka terminal atau command prompt di folder proyek Anda.

2. Gunakan perintah berikut untuk menghubungkan proyek ke repository Anda: 

        git init
        git remote add origin https://github.com/username/repository.git

3. Commit perubahan Anda dan upload dengan perintah: 

        git add .
        git commit -m "First commit"
        git push -u origin main

Langkah 3: Publish ke GitHub Pages

1. Buka halaman repository Anda di GitHub.

2. Klik Settings, scroll ke bawah, dan temukan bagian GitHub Pages.

3. Di bagian Source, pilih main branch dan klik Save.

4. Setelah beberapa saat, website Anda akan tersedia di URL: https://username.github.io/repository
