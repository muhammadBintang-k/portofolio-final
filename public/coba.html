Tutorial: Cara Membuat Form Login

Berikut adalah langkah-langkah untuk membuat form login sederhana menggunakan HTML dan Tailwind CSS:

  1. Buat struktur dasar form dengan elemen-elemen seperti `form`, `label`, `input`, dan tombol submit. 
                  
    <form action="/login" method="POST">
      <label for="username">Username:</label>
      <input type="text" id="username" name="username" required>
      
      <label for="password">Password:</label>
      <input type="password" id="password" name="password" required>
      
      <input type="submit" value="Login">
      <input type="reset" value="Reset">
    </form>
  2. Tambahkan atribut `action` pada elemen `
    ` untuk menentukan endpoint di mana data akan dikirim, dan gunakan metode `POST`. 
                  
    <form action="/login" method="POST">
      ...
    </form>
  3. Gunakan Tailwind CSS untuk memperindah form, dengan kelas-kelas seperti `w-full` untuk input yang memenuhi lebar container, dan `focus:outline-none` untuk efek fokus yang lebih interaktif. 
                  
    <input
      type="text"
      id="username"
      name="username"
      required
      class="w-full px-4 py-2 mb-4 border border-gray-300 rounded-md focus:outline-none focus:ring-2 focus:ring-button focus:border-transparent"
    />
  4. Terapkan styling pada tombol menggunakan Tailwind, dengan memberikan efek hover dan transisi agar tombol lebih menarik. 
                  
    <input
      type="submit"
      value="Login"
      class="px-4 py-2 bg-button text-white rounded-md hover:bg-hover transition duration-300 ease-in-out"
    />
  5. Jangan lupa tambahkan validasi dengan atribut `required` untuk memastikan pengguna mengisi data sebelum mengirim form. 
                  
    <input type="text" id="username" name="username" required>
    <input type="password" id="password" name="password" required>
